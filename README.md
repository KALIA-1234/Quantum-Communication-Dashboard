# ⚛ Quantum Communication Dashboard

A Flask-based interactive dashboard demonstrating quantum communication concepts using Qiskit.

## 🚀 Features

- ⚛ Bell State Simulation
- 🕸 Quantum Entanglement Simulation
- 🚀 Quantum Teleportation Simulation
- 📊 Measurement Results
- 📈 Quantum Circuit Visualization
- 📉 Quantum Histograms

## 🛠 Technologies Used

- Python
- Flask
- Qiskit
- Qiskit Aer
- Matplotlib
- NumPy
